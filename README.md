# Aplicações Práticas em Python
--------------------------------------------------------------------------------

Repositório com exercícios e aplicações básicas em Python, desenvolvidos durante meus estudos para praticar lógica de programação e conceitos fundamentais da linguagem.

Conteúdos:

*Entrada de dados (input)

*Conversão de tipos (int, float)

*Operações aritméticas

*Cálculos práticos (salário, desconto, média, conversões)

*Formatação de saída (f-strings)

*Código comentado e organizado
